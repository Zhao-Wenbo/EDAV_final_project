Data Analysis on Income Equality, Financial Indicator and Greenhouse Emission

Chapter 2 Data sources

A variety of information including economic, social welfare, education, greenhouse emissions, electricity production structure etc is needed to answer the questions we seek to answer. We use the comprehensive dataset of World Development Indicators from the world bank that collects various development indicators from officially-recognized international sources. It presents the most current and accurate global development data at national, regional and global estimates.

The data set contains several files:

  1. WDISeries.csv contains the information about what series are collected, with 1443 records and 20 columns. Each record contains the information for a series including its name, a short identifier code for it, the topic of the series, its source and so on.

  2. WDICountry.csv contains the information of countries and regions that are recorded, with 265 rows and 30 columns. Each row represents a country or region and presents its name, region, income group among several other identifiers.

  3. WDIData.csv contains data for all attributes from WDISeries.csv, for all the countries listed in WDICoutnry.csv. After cleaning the data and removing missing values, we use data from this file for every country and every series stated above. Since there is an extra Not classified class for countries, there are in total (265+1) * 1443 = 383838 records. Each row contains the country name and code, the series name and code, and the respective series value from 1960 to 2020. This constitutes a total of 66 columns altogether.

When matching the country name of WDICountry.csv and WDIData.csv we found that some names are not the same. There are two reasons that cause this difference:

  • WDICountry.csv uses country names with non-ASCII characters, such as Côte d'Ivoire and Curaçao while WDIData.csv transforms them into ASCII characters

  • There is a minor difference in presenting country groups. For example, WDICountry.csv uses IDA & IBRD while WDIData.csv uses IDA & IBRD countries.

We manually changed the names in WDICountry.csv to match those in WDIData.csv.
